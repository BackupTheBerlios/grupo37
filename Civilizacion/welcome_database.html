Desarrollo de aplicaciones de base de datos

Las funciones de desarrollo de aplicaciones de base de datos varían según la versión de JBuilder.

JBuilder proporciona una serie de funciones que permiten crear y mantener con facilidad aplicaciones de base de datos.

Los componentes DataExpress y dbSwing son componentes Borland de valor añadido. Se encuentran en la paleta de componentes del diseñador de interfaces de usuario, y se pueden arrastrar a la superficie de diseño. Estos componentes facilitan las operaciones de conexión con bases de datos, recuperación de datos de una fuente, guardado de los cambios en la fuente de datos y manipulación de los datos. También proporcionan compatibilidad internacional.

JDataStore proporciona funciones de base de datos a las aplicaciones utilizando un solo archivo JDataStore y el controlador JDBC de JDataStore (y las clases que lo aceptan). No se necesita ningún proceso de servidor para las conexiones locales. Además de la aceptación de la norma industrial JDBC, se puede aprovechar el aumento de la comodidad y la flexibilidad que supone el acceso a JDataStore directamente desde la API de DataExpress. Es posible utilizar ambos tipos de acceso en la misma aplicación.

El Explorador de bases de datos es un visualizador de bases de datos jerárquicas, desarrollado íntegramente en Java, que permite también presentar, introducir y modificar los datos. Muestra información basada en JDBC sobre los metadatos de la base de datos, dentro de una ventana con dos paneles. Para abrir el Explorador de bases de datos, seleccione Herramientas|Explorador de bases de datos.
